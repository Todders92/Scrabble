# _Scrabble_

#### _A console application testing a scrabble score for a word_

#### By _**Todd Walraven**_

## Description

_This application has tests for determining what scrabble score the user would get for an inputted word. It allows the user to input a word, the logic determines what the score would be and it is desplayed to the user through the console._
